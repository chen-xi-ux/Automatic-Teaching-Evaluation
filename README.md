# 自动化评教工具

基于 Python Tkinter 开发的图形界面评教管理工具。

## 📋 项目简介

本项目实现了一个具有友好用户界面的自动化评教管理工具，采用 Tkinter GUI 框架开发，提供直观的文件配置、权重管理、任务控制和日志显示功能。

## ✨ 主要功能

### 界面功能模块

#### 1. 文件配置区
- **评教任务编号文件**：配置评教任务编号列表文件路径
- **学号文件**：配置学生学号列表文件路径
- **教师名单文件**：配置特殊教师名单（自己人模式），支持开启/关闭开关

#### 2. 权重配置区
- **优秀权重**：调节优秀评价的选择权重（0-10）
- **良好权重**：调节良好评价的选择权重（0-10）
- **中等权重**：调节中等评价的选择权重（0-10）
- **应用配置**：将当前权重设置应用到系统

#### 3. 状态显示区
- 当前处理用户
- 当前处理课程
- 当前处理教师
- 评价结果

#### 4. 操作控制区
- **开始评教**：启动评教任务（演示模式）
- **暂停评教**：暂停正在执行的任务
- **终止评教**：停止当前任务
- **清空日志**：清除日志显示区域

#### 5. 日志显示区
- 实时显示系统运行状态
- 自动记录时间戳
- 支持滚动查看历史日志

## 🛠 技术栈

- **编程语言**：Python 3.x
- **GUI框架**：Tkinter（Python标准库）
- **多线程支持**：threading模块
- **消息队列**：queue模块

## 📦 系统要求

- Python 3.6 或更高版本
- 支持 Windows/Linux/macOS 操作系统
- 需要图形桌面环境

## 🚀 安装说明

### 1. 检查Python环境

确保已安装 Python 3.6 或更高版本：

```bash
python --version
```

### 2. 运行程序

直接运行主程序文件：

```bash
python main.py
```

### 3. 必需文件

程序运行时需要以下配置文件（与程序同目录）：

- `list.txt` - 评教任务编号列表（每行一个编号）
- `user.txt` - 学生学号列表（每行一个学号）
- `teacher.txt` - 教师名单列表（每行一个教师姓名，可选）

## 📖 使用指南

### 首次使用流程

1. **启动程序**：运行 `python main.py`
2. **阅读协议**：在弹出的用户协议对话框中等待3秒倒计时
3. **同意协议**：点击"同意"按钮进入主界面
4. **配置文件路径**：在文件配置区设置必要的文件路径
5. **配置权重**（可选）：根据需要调整评价权重
6. **开始使用**：点击相应按钮进行操作

### 功能操作说明

#### 文件配置
1. 在对应输入框中输入文件路径，或点击"浏览"按钮选择文件
2. "自己人模式"开关用于启用特殊教师名单功能

#### 权重配置
1. 拖动滑块调整各类评价的权重值
2. 点击"应用权重配置"按钮确认设置

#### 任务控制
1. **开始评教**：初始化任务并启动执行
2. **暂停评教**：临时停止任务执行（可继续）
3. **终止评教**：完全停止任务（不可继续）
4. **清空日志**：清除所有日志记录

## ⚠️ 注意事项

### 功能状态说明

本项目当前为**演示模式**：

- 所有业务功能按钮可正常点击和响应
- 按钮的视觉样式、交互行为完全保留
- 点击功能按钮将显示演示提示，不执行实际业务逻辑
- 10秒后自动恢复就绪状态

### 使用建议

1. **文件编码**：所有文本配置文件应使用 UTF-8 编码
2. **文件格式**：每行一个数据项，不要包含空行
3. **路径格式**：在 Windows 系统中注意使用正确的路径分隔符
4. **权限要求**：确保程序有读取配置文件的权限

### 安全提示

1. 妥善保管配置文件，避免泄露敏感信息
2. 定期更新教师名单文件内容
3. 不建议在多用户共享环境中使用

## 🔧 常见问题

### Q: 程序启动无反应？
**A:** 检查是否在命令行环境中运行，Windows系统可直接双击 `main.py` 文件

### Q: 文件浏览按钮无法打开对话框？
**A:** 确保在图形界面环境中运行程序

### Q: 日志显示乱码？
**A:** 检查配置文件编码格式是否为 UTF-8
